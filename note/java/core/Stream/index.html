使用流操作

stream还可以并行处理和计数

流和集合的差异：

  1. 流并不存储元素
  2. 流操作不会修改原始数据
  3. 流操作时惰性执行。即直到需要其结果，操作才会执行，因此我们可以操作无限流。

典型流程

如下所示，流的操作：

  1. 创建一个流

  2. 将初始流转换为其他流的中间操作，可以有多个步骤

  3. 终止操作，产生结果。这个操作会执行之前的惰性操作，这个流结束。
