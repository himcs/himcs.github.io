路由

作用

用来控制URL与页面的对应,以及何时跳转

Vue Router

基本用法

使用<router-view>来显示路由页面，使用<router-link>来跳转路由

项目用法

顶级路由

  • 后台主界面
  • 登录页
  • 错误页
  • 中转页面

后台主界面由

后台主界面又是一层路由,定义为Layout组件

两个组件都是用了Layout默认会复用Layout（布局组件）

提醒一下，当使用路由参数时，例如从 /user/foo 导航到 /user/bar，原来的组件实例会被复用。因为两个路由都渲染同个组件，比起销毁再创建，复用则显得更加高效。不过，这也意味着组件的生命周期钩子不会再被调用。

动态路由

即经过权限控制后，动态修改路由

核心方法

router.addRoutes(asyncRoutes);

权限控制

每次经过路由，验证权限

通过router.beforeEach()进行控制
