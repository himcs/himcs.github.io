# TF-IDF 算法

ES5前使用。

  • TF（Term Frequency): 检索词在一篇文档中出现的频率
  • DF（Document Frequency): 检索词在所有文档出现的频率
  • IDF(Inverse Document Frequency)
    • IDF = log(全部文档数/检索词出现过的文档总数)
    • 词条权重

TF-IDF 公式

(TF(WORD)*IDF(WORD))
