# SVN 提交后自动更新测试服务器

原理

​ 提交后触发服务器的 post-commit hook

​ svn服务器调用更新脚本或者更新URL

# PHP 执行SVN UP

$username = 'svn用户名';
$password = 'svn密码';
$path = '更新目录';
//echo t 接受不安全的连接
exec("export LANG=\"en_US.UTF-8\";echo t | /usr/bin/svn up --username $username --password $password  --no-auth-cache $path 2>&1",$output);
print_r($output);
