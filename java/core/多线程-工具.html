# 进程与线程

进程是操作系统运行程序的基本单位

线程是进程划分成的更小的运行单位, Java线程共享进程的堆和方法区资源, 每个线程有自己的程序计数器,虚拟机栈和本地方法栈.

# 线程安全问题

当多个线程并发执行一条代码路径或并发修改类中属性,如果会产生不可预料的后果,这种问题就是线程安全问题. 常见的问题是Check-Then-Act操作，即通过检测一个可能失效的观察结果决定下一步的动作。

条件为 文件X不存在

操作为 创建文件X

检测成立，然后进行相关的操作，但事实上，得到文件X不存在和创建文件X之间，观察结果可能变得无效（例如另一个线程创建了文件X），从而会导致各种问题（文件覆盖，文件被破坏等等）
