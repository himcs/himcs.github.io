# 多线程-原理

Java多线程的锁是基于对象的,每个对象都可以作为一个锁.

# synchronized关键字

对象锁是可重入互斥锁,即只有一个线程可以持有这把锁,可以在方法上使用,也可以在代码块上使用

//锁this
public synchronized void testIntrinsicLock() {
        count++;
        System.out.println(count);
}

# 锁类型

一个对象有四种锁状态:

  1. 无锁
  2. 偏向锁
  3. 轻量级锁
  4. 重量级锁

无锁就是没有对资源进行锁定，任何线程都可以尝试去修改它。
