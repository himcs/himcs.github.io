# 修正

使用PdgCntEditor打开pdf，在编辑窗口复制处理后的目录。

  2. 页码偏移修正

# 格式修正

格式修正为以下格式

第16章　Java内存模型	291 
	16.1　什么是内存模型，为什么需要它	291
		16.1.1　平台的内存模型	292
		16.1.2　重排序	292

# 页码偏移修正

页码是书籍的页码可能与PDF不对应，找到相应的差值，在编辑窗口点击+-号修正。

也可以通过编程的方式修正。
